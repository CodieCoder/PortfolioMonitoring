
## Description

Back-end built for monitoring my personal portfolio website and contact form.
Built on [Nest](https://github.com/nestjs/nest) framework using Typescript

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
